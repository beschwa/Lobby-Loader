# Lobby Loader
An advanced game lobby app where users can create the age-old 'game lobby' for games they love and meet new players!

## Technologies
Front End
	- React.js
	- React-Redux
	- Semantic UI
	
Back End
	- Rails
	- Active Record
	- PostgreSQL

Other
  - JWT Tokens for Auth
  - IGDB API for populating games
